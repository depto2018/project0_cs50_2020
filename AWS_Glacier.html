AWS Glacier

Amazon S3 Glacier and Glacier Deep Archive are a secure, durable and a low cost cloud solution for data archiving and long-term backup. It means data that does not need to be accessed every day. It costs around $1 per terabyte per month.

Retrieval methods

Expedited retrievals return data in 1-5 minutes, great for Archive use cases. Standard retrievals completed between 3-5 hours, and work well for less sensitive data like backups data, media editing, long-term analytics. Bulk retrievals are the lowest method returning large ammount of data between 5-12 hours. Amazon S3 Glacier deep archive provides two retrievla options ranging from 12 to 48 hours.

Durability and Scalability

Amazon S3 Glacier and Glacier Deep Archive are designed for 99.999999999% of durability. Data is automatically distributed across a minimum of 3 AZs that are geographically separated within an AWS Region.

Security and Compliance

AWS Glacier storage classes sopport three different forms of encryption. These storage classes also support security standards and compliance certifications including SEC Rule 17a-4, PCI-DSS, HIPAA/HITECH, FedRAMP, EU GDPR, and FISMA, and Amazon S3 Object Lock enables WORM storage capabilities, helping satisfy compliance requirements for virtually every regulatory agency around the globe.

Archives

Data in AWS Glacier is stored in archives which would be an object in AWS S3. An archive can be any kind of file (it can be tar or zip) as large as 40 terabytes. The content of an arcchive is inmmutable, which means after the archive is created it can not be updated. You can store an unlimited ammount of data in Glacier.

Vaults

Vaults are the containers to store the archives. It can be seen as the buckets for AWS S3. You can perform a variety of operations from the management console or SDKs like create a vault, delete vault, lock vault,list vault metadata, retrieve vault inventory, tag vaults and configure vault notifications. Under a single AWS account, you can have up to 1,000 vaults.

Amazon S3 Glacier select

Amazon S3 Glacier Select allows queries to run directly on data stored in Amazon S3 Glacier without having to retreive the entire archive. Amazon S3 Glacier Select changes the value of archive storage by allowing you to process and find only the bytes you need out of the archive to use for analytics.

Amazon Snowball and Diect Connect

Amazon Snobball and Direct connection are two methods you can use to migrate or send your data from your on-premises network to the cloud in a fast and secure way.

Vault Lock

Amazon S3 Glacier Vault Lock allows you to easily deploy and enforce compliance controls on individual S3 Glacier vaults via a lockable policy. You can specify controls such as “Write Once Read Many” (WORM) in a Vault Lock policy and lock the policy from future edits.

Encryption by default

Amazon S3 Glacier automatically encrypts data at rest using Advanced Encryption Standard (AES) 256-bit symmetric keys and supports secure transfer of your data over Secure Sockets Layer (SSL).
