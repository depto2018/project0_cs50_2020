AWS S3 (Simple Storage Service)

What is AWS S3?

AWS S3 is an object (file) level storage service which can be used to save general purpose data on the cloud and access it from anywhere. It can be seen as a NAS (Network Attached Storage) in an on-premise network. General features of AWS S3 are the following:

  • Data is distributed across at least three AZs with the exception of 1A class S3 services where the data is kept in one AZ only, which makes it a cheaper solution.
  • It supports encryption.
  • Provides automatic data classification. For example if data is not manipulated in any way during a certain period of time, it can be automatically moved to AWS Glacier for long term storage.
  • Useful for big data analytics.

How to get data into S3?

There are different ways of getting data into S3 as follows:

  • Uploading the files manually to an S3 bucket from your local desktop like the way you do with a storage cloud service like google drive or dropbox.
  • Files can be uploaded through an API (Application Programming Interface).
  • Amazon Direct Connect can be used to upload files through a VPN.
  • An Storage Gateway can also be used to synchronize local data into AWS S3.
